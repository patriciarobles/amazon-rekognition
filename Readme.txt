Este proyecto utiliza Amazon Rekognition para el reconocimiento facial. Se realizó en noviembre del 2019 y no está actualizado desde entonces. 

1. Configuración cuenta Amazon Rekognition en el archivo app.php, líneas 22 a la 25

2. Desde el archivo index.php, si se accede desde un móvil, se hace una foto a una persona y se detecta si esa persona está en la imagen img/target.png (cambiar)
